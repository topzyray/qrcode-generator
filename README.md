# Qrcode-generator
URL QR code generator written in Node.js using inquirer, qr-image and fs modules

# Usage
1. Install Node on your PC.
2. Install VS Code
3. Clone the repository
4. Open a new terminal on your VS Code
5. Run "npm install" to install the package used for the project
6. Crosscheck package installation in the package.json file
7. Run "node index.js" on your terminal
8. Enter the full URL you which to generate the QR code
9. Boom! Check your directory for the QR image
